# Node.js starter

## Usage

```sh
$ npx degit weiran.zsd/starter
```

```sh
# run linting
$ npm run lint [-- --fix]

# run test
$ npm test

# watching test
$ npm run test:watch

# release
$ npm run release
```

## Features

* Node.js esm
* prettier
* eslint + eslint-plugin-node
* pre-commit with husky
* release with standard-version
* testing with jest
* gh actions
